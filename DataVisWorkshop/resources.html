Resources for Data Visualisation

From the 4th ESLR Workshop

July 2021

What these resources are for

This is a list of resources that might be useful as you learn your way around data visualisation, particularly in R using the tidyverse and ggplot. They are from a short 2-hr data visualisation workshop, which focused an starting out with ggplot and a relatively simple dataset. The slides and full exercise from the workshop are included in the resources listed below; they are designed to be done independently if you weren't able to attend the live workshop.

Given the brevity of the workshop, the aim is for people to go out into the Google and dive into new resources on their own to advance their practice. This list will provide a starting point, but it is only selection of the vast array of resources available out there.

Resources

Slides from the workshop in PDF format and completed annotated RMarkdown Notebook.

WEB DuBois' hand drawn data visualistions from the 1900 World's fair, and Martina Pugliese's DoodleDat Card, hand drawn data visualisation.

An introduction to RMarkdown (documents, presentations and notebooks in R)

R documentation

General Tidyverse documentation; specific tidyr and ggplot2 packages.

DescTools package manual; a package for descriptive statistics.

Work on visual metaphors in interpreting graphs.

Woodin, G., Winter, B., & Padilla, L. (2021). Conceptual metaphor and graphical convention influence the interpretation of line graphs. IEEE transactions on visualization and computer graphics

.

Information is beautiful, for general inspiration and beautiful datavis.

Coolors colour palette generator (note that this has a filter for various kinds of colourblindness; you can see how your palette will look to a colourblind person).

R Colourbrewer, a package for generating palettes within R. For more interactive palette generation, Coolors is great.

A detailed guide on visulasing model residuals in R

We didn't cover much visualisation related to inferential statistics, focusing mainly on descriptive visualisation. However, especially if you often use mixed effects modeling, this primer is well worth a look.

A detailed workshop/tutorial in network visualisation in R, using a variety of packages, from Katya Ognyanova

We also didn't have much time to get in depth with network visualisation, but this is a great (and very detailed) notebook from a longer workshop dedicated to the topic.

threejs, an R package for using the powerful data visualisation tool d3js via R. Great for advanced and interactive data visualisations.
